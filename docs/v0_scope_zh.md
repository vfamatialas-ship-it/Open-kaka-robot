# V0 范围说明

V0 是基础调试版，目标是把已经实机跑通的内容整理成一个干净、可上传、可复现的开源仓库。

## V0 包含

- 主臂读取
- 从臂读取
- 零点管理
- 关节限位
- 单关节测试
- 多关节遥操作
- 安全 disable / 急停状态
- 上位机调试工具
- 粉色从臂 URDF/STL
- 网页 3D 可视化
- 主臂控制虚拟从臂映射调试

## V0 不包含

- LeRobot 数据集采集
- VLA 推理
- Paxini 触觉传感器
- Orbbec Gemini335 深度/IR/IMU SDK
- 双臂协同
- 运动学/动力学库 Pinocchio
- 完整重力补偿模型

## 后续版本建议

```text
V0.1: 单臂主从遥操作稳定版
V0.2: raw episode 数据采集
V0.3: LeRobot 数据格式转换
V0.4: 双臂遥操作
V0.5: 相机/手眼标定
V0.6: 触觉传感器接入
V1.0: 可复现实验任务和数据集
```
