# 安全说明

Open Kaka V0 可以向真实达妙电机发送运动命令。使用前请确认：

1. 机械臂周围没有人员和易碰撞物体。
2. 电源、急停、断电方式都可以立即触达。
3. 已经读取当前关节位置。
4. 已经确认零点和软件限位。
5. 首次运动使用单关节、小范围、低速度测试。

## 默认安全设计

- 读取脚本默认只读。
- 运动脚本必须添加 `--enable-motion YES`。
- 保存硬件零点和硬件限位需要二次确认。
- 遥操作目标会经过软件限位检查。
- Ctrl+C 会尝试 disable 粉色从臂。

## 不要做的事

- 不要一上电就回零。
- 不要在没有限位文件时运动。
- 不要把主臂大幅快速甩动后直接控制从臂。
- 不要在不知道关节方向时全关节遥操作。
- 不要把真实采集数据、日志和本地串口信息直接提交到公开仓库。
